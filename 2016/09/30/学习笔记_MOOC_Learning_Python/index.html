MODULE 01: Python基础之第二周 Python面面观

题目内容：

找第n个默尼森数。P是素数且M也是素数，并且满足等式M=2P-1，则称M为默尼森数。例如，P=5，M=2P-1=31，5和31都是素数，因此31是默尼森数。

输入格式:用input()函数输入，注意如果Python 3中此函数的返回类型
输出格式：int类型

输入样例：4
输出样例：127
